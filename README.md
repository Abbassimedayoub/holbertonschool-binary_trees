# Binary Trees — C

## Overview

Implementation of binary trees in **C**, from node creation to property analysis. Every function is written from scratch (no external library), documented, and compliant with the **Betty** coding style. Pair project from the Holberton School curriculum.

## Implemented Functions

| Category | Functions |
|---|---|
| Construction | `binary_tree_node`, `binary_tree_insert_left`, `binary_tree_insert_right`, `binary_tree_delete` |
| Checks | `binary_tree_is_leaf`, `binary_tree_is_root`, `binary_tree_is_full`, `binary_tree_is_perfect` |
| Traversals | `binary_tree_preorder`, `binary_tree_inorder`, `binary_tree_postorder` |
| Measurements | `binary_tree_height`, `binary_tree_depth`, `binary_tree_size`, `binary_tree_leaves`, `binary_tree_nodes`, `binary_tree_balance` |
| Family | `binary_tree_sibling`, `binary_tree_uncle` |

## Usage

```bash
gcc -Wall -Werror -Wextra -pedantic *.c -o binary_trees
```

Each `.c` file implements one function; headers and helper printers are provided by the project.

## Skills Demonstrated

Recursive tree algorithms, pointer-based data structures, memory management, and collaborative development (pair programming, branches and merges).
